# Партнёрские программы — инструкция по подключению

Данные проверены по официальным страницам программ 6 июля 2026. В интернете много устаревших цифр — доверяй этому файлу и тому, что видишь в кабинете программы при регистрации.

## Как это работает на сайте

1. Пайплайн автоматически помечает новости, где упоминаются отслеживаемые AI-инструменты (`scripts/tools-list.mjs`).
2. Ты регистрируешься в программе → получаешь свою реферальную ссылку.
3. Вставляешь ссылку в `config/affiliates.json` в поле `url` нужного инструмента.
4. После следующего деплоя у всех новостей с упоминанием инструмента появится плашка «Try X →» с твоей ссылкой и дисклеймером. Пока `url` пустой — ничего не показывается.

Деньги приходят, когда читатель кликает и оформляет подписку на инструмент.

## НЕ трать время на эти (проверено)

- **OpenAI/ChatGPT, Anthropic/Claude, Google/Gemini** — потребительских партнёрок НЕ существует (только enterprise-программы без комиссий)
- **Notion** — программа закрыта для новых участников (проверяй раз в квартал)
- **HeyGen** — теперь только для блогеров с 5000+ подписчиков в соцсетях, сайты исключены
- **Copy.ai** — страница программы отдаёт 404, программа фактически брошена
- **Midjourney, Runway, Suno** — официальных партнёрок нет
- **Descript, Grammarly, Surfer SEO** — работают, но платят разово (не регулярно) — низкий приоритет

## Что вписывать в config/affiliates.json

После регистрации в личном кабинете программы будет «Your referral link» вида `https://try.elevenlabs.io/xxxxx` или `https://partnerstack.com/...`. Скопируй её в поле `url` соответствующего инструмента:

```json
{
  "keyword": "ElevenLabs",
  "label": "ElevenLabs",
  "url": "https://try.elevenlabs.io/ТВОЯ-ССЫЛКА",
  "disclosure": "Affiliate link — we may earn a commission at no extra cost to you."
}
```

Затем закоммить и запушь (или попроси Claude) — плашки появятся после автодеплоя.
